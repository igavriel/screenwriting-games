# screenwriting-games
Screenwriting for digital games

GitHub Pages: https://igavriel.github.io/screenwriting-games/

to add a new game, add the game to the `js/games.js` file.

the game should be a object with the following properties:
- title: the title of the game
- href: the href of the game
- cover: the cover of the game
- accent: the accent color of the game

the cover should be a relative path to the cover image, or an empty string if no cover is needed.
the accent should be a hex color.
